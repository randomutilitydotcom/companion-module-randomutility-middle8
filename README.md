# companion-module-randomutility-middle8

Bitfocus Companion module to control **RandomUtility Middle8** via UDP / TCP / OSC.

See `companion/HELP.md` for user-facing documentation.

## Development

```bash
corepack enable
yarn install
yarn dev
```

## Build/package

```bash
yarn build
# or
yarn companion-module-build
```

This produces `pkg/` and `pkg.tgz` which can be imported into Companion.
